Description:

fsm.py is a script which manages a state machine which represents the actions that a robot must do in order to play Cluedo in a fictitious environment. The FSM has 5 states: Navigation: the robot navigates through an environment thanks to the random position which has been given by a custom service, possible outcomes are:

  • investigation
  • navigation Investigation: the robot investigates thanks to armor client, in order to know if there is a consistent hypothesis, possible outcomes are:
  • navigation
  • investigation
  • go_to_oracle GoToOracle: the robot goes to the oracle; even if in this implementation the only outcome is assert, in theory the possible outcomes are:
  • navigation
  • investigation
  • go_to_oracle_
  • assert Assert: the robot asks if among the consistent hypotheses it has there is the correct one, possible outcomes are:
  • navigation
  • investigation
  • go_to_oracle
  • assert
  • finished Finished: the robot has won
