fsm.\+py is a script which manages a state machine which represents the actions that a robot must do in order to play Cluedo in a fictitious environment. The F\+SM has 5 states\+: Navigation\+: the robot navigates through an environment thanks to the random position which has been given by a custom service, possible outcomes are\+:
\begin{DoxyItemize}
\item investigation
\item navigation Investigation\+: the robot investigates thanks to armor client, in order to know if there is a consistent hypothesis, possible outcomes are\+:
\item navigation
\item investigation
\item go\+\_\+to\+\_\+oracle Go\+To\+Oracle\+: the robot goes to the oracle; even if in this implementation the only outcome is assert, in theory the possible outcomes are\+:
\item navigation
\item investigation
\item go\+\_\+to\+\_\+oracle\+\_\+
\item assert Assert\+: the robot asks if among the consistent hypotheses it has there is the correct one, possible outcomes are\+:
\item navigation
\item investigation
\item go\+\_\+to\+\_\+oracle
\item assert
\item finished Finished\+: the robot has won 
\end{DoxyItemize}
