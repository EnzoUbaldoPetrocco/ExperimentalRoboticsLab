\item investigation

\item navigation
\item investigation
\item go\+\_\+to\+\_\+oracle\+\_\+
\item assert Assert\+: the robot asks if among the consistent hypotheses it has there is the correct one, possible outcomes are\+:
\item navigation
\item investigation
\item go\+\_\+to\+\_\+oracle
\item assert
\item finished Finished\+: the robot has won
\end{DoxyItemize}
